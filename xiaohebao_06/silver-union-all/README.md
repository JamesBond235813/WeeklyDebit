# 项目说明

## 项目概览

- 本仓库为多模块 Spring Boot（Java 17）后端工程，面向客户信息与业务流转（分配、跟进、归属等）管理。
- 模块划分：
  - `silver-union-ops`：主业务服务（Spring Boot 应用）。
  - `silver-union-common`：通用组件（异常、工具类、校验、web 处理等）。
  - `silver-union-z-generator`：MyBatis-Plus 代码生成器。

## 技术栈

- Java 17 / Spring Boot 3.4.3 / Spring Cloud 2024.0.0
- Spring Web / Spring Security / JWT / Validation / WebSocket / Actuator
- MyBatis + MyBatis-Plus / PageHelper / Spring Data JDBC
- MySQL 8（驱动）/ Knife4j OpenAPI / Lombok / MapStruct

## 本地运行

- 默认使用 **default profile**（不指定 `spring.profiles.active` 即可）。
- 本地运行时，先执行 `source ~/.zshrc`, 再执行`j17`, 确保切换到 jdk17 + maven 3.9.9环境
- 推荐启动方式：
  - `./mvnw -pl silver-union-ops -am spring-boot:run`

## 配置说明

- 运行配置位于 `silver-union-ops/src/main/resources/`（如 `application.yml` / `application-*.yml`）。
- 本地开发默认 profile（default）。

## 代码生成器（silver-union-z-generator）

- 直接新增一个带 `main` 方法的类并运行即可。
- 示例：`com.jhl.silver.union.MybatisPlusGeneratorCustomer`

## 代码规范

- 未配置 Checkstyle/Spotless 等格式化工具。
- 统一以当前 IntelliJ IDEA 的格式化设置为准。

## 功能清单

### 使用说明

- 超管可可将客户信息分配到企业公海、部门公海以及特定员工
- 部门管理员可将本部门公海的客户信息 分配给本部门以及下属部门的员工
- 员工可将客户信息退回到归属部门的公海
- 将客户信息分配给员工时， 只能分配给处于【在线】状态的员工
- 在线状态由用户/员工手动维护，系统不干涉。

### 说明事项

- 未在备注的情况下，【删除】均为【逻辑删除】
- 客户信息模块
    - 客户列表查询条件
        - 跟进时间：时间区间查询
        - 客户电话：精确查询
        - 申请时间：时间区间查询
        - 客户名字：左匹配查询
        - 客户星级：下拉框选择查询
        - 跟进状态：下拉框选择查询
        - 部门：下拉框选择查询，可多选
    - 修改 用户手机号、用户姓名的操作，仅超管有权限
    - 关于时间轴（历史记录）分两类：
        - 客户数据变化的历史记录（不包含【跟进记录备注】）。
        - 【写跟进记录】 仅修改客户信息中的跟进信息
    - 客户各属性说明
        - 客户分组，即一星，二星，三星。需要支持自定义客户分组
        - 单独设置【申请时间】，即为获取数据的时间（不一定是入库时间），从业务角度出发， 申请时间越晚，数据质量越高。
        - customer_info_item【mobile】 必须为中国大陆手机号
        - customer_info_item【call_tips】 枚举类型需要扩充（亮）
        - customer_info 列表排序 固定以 最后跟进时间， 申请时间降序排列
    - 关于上级部门人员对数据的操作权限
        - 上级可以将数据调给某个特定的人（可跨层级调）
        - 上级可以在【跟进记录备注】中追加评语， 但不可修改其它字段
    - 业务员对客户数据的操作权限
        - 业务员可将数据退回公海（归属部门）
        - owner_favorite用于表示业务员将客户信息纳入【我的客户】进行管理。
        - 优先做人工分配客户信息给到业务员
    - 业务字典信息
        - 业务字典可自定义。 可进行增加、修改， 但不可删除，若是不再使用特定字典项， 将其置为【禁用】
        - 电话标签新增： 空号；停机；关机；长响不接；拒绝挂断；接通-听贷挂断；接通-稍后联系；接通-无意向；有意向-补充资料；
